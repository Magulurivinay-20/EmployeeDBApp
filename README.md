# Employee JDBC App

## Description
This Java console app performs CRUD operations on a MySQL employee database using JDBC.

## Technologies
- Java
- JDBC
- MySQL
- VS Code

## How to Run
1. Set up MySQL DB
2. Update credentials in `DBConnection.java`
3. Compile and run using:
   - `javac -cp "lib/*" src/*.java`
   - `java -cp "lib/*:src" EmployeeCRUD`
